Defintion

HTML headings are titles or subtitles that you want to display on a webpage.

Knowledge Is Power

H1 is an HTML tag that identifies the first heading on a page. It is used as the name of the page or publication. H1 tag example.

Knowledge Is Power

An H2 tag marks the first sub-heading after your document's main heading. It defines the second-level headings on your webpage.

Knowledge Is Power

The H3 element defines a level-three heading. This heading is more important than an H4 but less important than an H2.

Knowledge Is Power

The H4 element defines a level-four heading. This heading is more important than an H5 but less important than an H3. The deprecated ALIGN attribute suggests the horizontal alignment for the heading on visual browsers. Possible values are left, right, center, and justify.
